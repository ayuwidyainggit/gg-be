# Skills

- `@orchestrator` — default router and integrator.
- `@artifact-planner` — writes scoped plans for multi-step or evidence-heavy work.
- `@explorer` — discovery, file mapping, and repo pattern lookup.
- `@fixer` — bounded code changes, tests, and refactors.
- `@designer` — UI/UX ownership when relevant.
- `@architect` / `@oracle` — architecture, tradeoff, and review lanes.
- `@librarian` — current docs and external API/library research.
- `@quality-gate` — final conformance and risk signoff.

Use this with `AGENT_ROUTING.md`; keep ownership explicit so work does not collapse into `@orchestrator`.
