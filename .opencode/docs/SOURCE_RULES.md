## How to use this audit

- When updating `AGENTS.md` or `.opencode/docs/`, keep this
  file in sync so the import provenance is preserved.
- When removing an external tool's rules file, re-run
  `--apply` to drop the corresponding entries here.
- When re-syncing external tools (after OpenCode-side rule
  changes), re-run with `--forward-to claude,codex,cursor`
  to write harmonized mirrors back to the external files.
