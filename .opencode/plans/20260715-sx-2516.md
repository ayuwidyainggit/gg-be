## Requirements

1. **Replace scope**: `distinct (cust_id, ro_date)` dari baris valid. Customer/tanggal lain TIDAK terhapus. Order `is_sales_mapping=false` TIDAK terhapus.
2. **Atomic transaction**: `delete detail JOIN header scope` → `delete header scope` → `insert header` → `insert detail` dalam SATU `WithinTransaction`. Gagal satu step → rollback seluruhnya.
3. **Lock**: `pg_advisory_xact_lock(hashtextextended(cust_id||':'||date, 0))` per scope row; `SELECT 1 FROM sls.order WHERE cust_id=$1 AND is_sales_mapping=true AND ro_date = ANY($2::date[]) FOR UPDATE` setelah advisory.
4. **Detail first**: delete `sls.order_detail` sebelum `sls.order` (FK mungkin RESTRICT; tidak andalkan CASCADE).
5. **7-day rule** di parser (sebelum lookup master):
   - `document_date` > today+1 → error `"Transaction Date cannot be later than the current date."`.
   - `document_date` < today-7 → error `"Transaction Date cannot be more than 7 days before the current date."`.
   - `today-7` ≤ `document_date` ≤ `today` → lolos.
   - Timezone: Asia/Jakarta (`time.Now().In(jakartaLoc)`).
6. **ro_no reuse**: `ro_no` = `DocumentNo`; insert ulang dengan ro_no sama (delete-then-insert).
7. **Audit defaults**: `data_status=6`, `data_source=3`, `is_del=false`, `is_closed=false`, `validate_stok=false`, `is_paid_off=false`, `is_proforma_inv=false`, `pay_type=1`, `disc=0`, `is_sales_mapping=true`, `created_by=userId`, `created_at=now()`. Sudah ada di parser/Store existing; reuse.
8. **Mapping detail** (sudah ada di `parseImportOrders`/`Store`; hanya memastikan reuse):
   - `pro_id` = `parent_pro_id` parent principal.
   - Unit slot distributor → `qty1/qty2/qty3` + `sell_price*`, slot lain NULL.
   - `qty`/`qty_final` = total unit terkecil.
   - `unit_id*`/`conv_unit*`/`purch_price*`/`sell_price_system*` dari parent sesuai slot.
9. **Reuse**: `parseImportOrders`, `matchUnitSlot`, `mapQtyAndPriceToSlot`, `isImportedOrder`, `importDataStatus`, `importDataSource`, `WithinTransaction`, `model(ctx)`/`extractTx`. TIDAK duplikasi.
10. **Parser header contract**: 16 kolom `orderImportHeaders`. Sheet tanpa header match → error.

## Risks

- R1: FK `sls.order_detail` mungkin RESTRICT, bukan CASCADE. Mitigasi: delete detail dulu, JOIN ke header scope. Verifikasi saat implementasi.
- R2: Concurrent replace scope sama → race. Mitigasi: `pg_advisory_xact_lock(hashtextextended(cust_id||':'||date, 0))` per scope row.
- R3: 7-day rule timezone drift. Mitigasi: `time.Now().In(jakartaLoc)`; konstanta `var jakartaLoc = time.FixedZone("Asia/Jakarta", 7*3600)` di test agar deterministik.
- R4: Mock repo di test existing tidak signature-match. Mitigasi: extend interface dengan method yang ada implementasi, atau test langsung di service-level.
- R5: Revert requirement. Source changes di luar slice (controller/DTO/migration/staging/history) WAJIB direvert oleh executor sebelum claim. Planner merekam instruction; fixer mengeksekusi.
